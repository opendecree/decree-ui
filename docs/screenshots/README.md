# Screenshots

Generated by `scripts/screenshots.ts` for use in READMEs and docs.

## Regenerate

1. Start the decree backend with fixtures loaded:

   ```sh
   # from the decree/ repo
   make dev-stack  # or: scripts/dev-stack.sh up && scripts/dev-stack.sh seed
   ```

2. Start the UI dev server:

   ```sh
   # from decree-ui/
   npm run dev
   ```

3. Run the screenshot script:

   ```sh
   npm run screenshots
   ```

PNGs land in this directory — one per page, per color scheme (`light-*` and `dark-*`).

## Contract

The script looks up fixture IDs by `name` via `GET /v1/schemas` and `GET /v1/tenants`, so the
committed fixture names (`billing`, `showcase`, `acme`, `demo`) must stay stable. Renaming a
fixture means updating `scripts/screenshots.ts` in the same change.

## Environment

| Variable   | Default                 | Purpose                |
| ---------- | ----------------------- | ---------------------- |
| `BASE_URL` | `http://localhost:5174` | UI dev server URL      |
| `API_URL`  | `http://localhost:8080` | decree backend URL     |
| `OUT_DIR`  | `docs/screenshots`      | PNG output directory   |
